Documentation Webhooks

If you have specified a webhook url (or multiple), the LiquipediaDB API will send you pings if there is updated content or a page is moved, deleted or purged.

The following event types will be sent:

  • delete
  • edit
  • move
  • purge

Edit, delete & purge events

For edit, delete and purge events, the webhook will receive a POST call with a json object as content that looks as follows:

{
    "page": "Page Name",
    "namespace": 0,
    "wiki": "pathofwiki",
    "event": "event"
}

A newly created page will emit an edit event.

Move events

For move page event, the webhook will receive a POST call with a json object as content that looks as follows:

{
    "from_page": "Old Page Name",
    "page": "New Page Name",
    "from_namespace": 0,
    "namespace": 0,
    "wiki": "pathofwiki",
    "event": "move"
}

Wiki

All wiki names can be found in the API documentation. The wiki is relevant to know which wiki the update is from.

Page name & namespace

This page name and namespace can be used in normal API calls by setting the [[pagename::Page Name]] AND [[namespace::0]] in the conditions.

The namespace value usually has a different meaning per wiki, only a few select values are the same across wikis. The most important ones are 0 and -10.
The main public namespace is 0 in all wikis. Namespace -10 is the teamtemplate namespace for all wikis. Other values, like those for users private spaces will be different per wiki
